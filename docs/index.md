## Commands

### Generating the docs

Use [properdocs](https://properdocs.org/) structure to update the documentation.

> properdocs is a drop-in replacement for the unmaintained MkDocs.

Build locally with:

```
properdocs build
```

Serve locally with:

```
uv run properdocs serve
```
